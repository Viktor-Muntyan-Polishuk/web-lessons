Основні властивості CSS для роботи з текстом

  • Властивість font-size
  • Властивість font-weight
  • Властивість font-style
  • Властивість font-family
  • Властивість color
  • Властивість text-transform
  • Властивість text-decoration

Властивість font-size

Властивість font-size встановлює розмір шрифту.

Розмір шрифту - це висота від базової лінії літери до її верхнього краю.

Синтаксис:

CSS

 p {
    font-size: 20px;
 }

HTML

<p>Hello,world!!</p>

Результат:

Hello,world!!

Властивість font-weight

Властивість font-weight встановлює товщину шрифта.
Вага шрифтів, доступних вам, залежить від сімейства шрифтів, який ви використовуєте.
Деякі шрифти мають обличчя символів лише для нормальної (normal) та жирної (bold) товщини.
Для таких шрифтів жирність 100-500 - normal , а 600-900 - bold.

Синтаксис:

CSS

  p {
    font-weight: normal ;
    }
   a{
    font-weight: bolder ;
    }

HTML

<p>Hello, </p>
<a> world!!</a>

Результат:

Hello,

world!!

Властивість font-style

font-style це CSS-властивість визначальне яким чином шрифт має бути стилізований, чи це normal,italic , чи oblique.

Синтаксис:

CSS

one {
    font-style: normal ;
    }
two {
    font-style: italic ;
    }
    three {
    font-style: oblique 40deg ;
    }

HTML

<p> <a class="one">one</a>, <a class="two">two</a>, <a class="three">three</a>. </p>

Результат:

one, two, three.

Властивість font-family

font-family визначає пріоритетний список із однієї або декількох назв
сімейства шрифтів та/або загальне ім'я шрифту для вибраного елемента.

Синтаксис:

CSS

one {
    font-family:  Papyrus, fantasy ;
    }
    two {
    font-family:  "Apple Chancery", cursive ;
    }

HTML

<p> <a class="one">Papyrus</a>, <a class="two">Apple Chancery</a>. </p>

Результат:

one, two.

Властивість color

Модуль color детально описує значення, які дозволяють авторам визначати значення якості color.

Синтаксис:

CSS

one {
    color:red;
    }

HTML

<p> <a class="one">I</a> go to <a class="one">Hell</a>. </p>

Результат:

I go to Hell.

Властивість text-transform

Властивість text-transform вказує, як використовувати великі літери в тексті елемента.
Його можна використовувати, щоб текст відображався у верхньому чи нижньому регістрі або з великими літерами.

Синтаксис:

CSS

one {
    text-transform: uppercase;
    }
two {
    text-transform: lowercase;
    }

HTML

<p> <a class="one">Hello</a> and <a class="two">Hello</a>. </p>

Результат:

Hello and Hello.

Властивість text-decoration

Властивість скорочення text-decoration встановлює вигляд декоративних ліній у тексті.
Це скорочення від text-decoration-color , text-decoration-style та інші.

Синтаксис:

CSS

one {
    text-decoration-style: double;
    text-decoration-color: #21ff21;
    text-decoration: underline;
    }

HTML

<p> <a class="one">Hello</a>, world!! . </p>

Результат:

Hello, world!!.
