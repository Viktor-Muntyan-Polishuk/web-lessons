Розмітка заголовків

<h1> - html код головного тега, зазвичай використовується як заголовок всієї сторінки на самому початку змісту.

<h2> - 2-й за значимістю заголовок (html код), зазвичай використовується для поділу за змістом кілька частин змісту сторінки.

<h3> - 3-й за значимістю заголовок сторінки (html код), як правило, використовується як підзаголовок тега <h2>.

<h4>, <h5> та <h6> - 4, 5 і 6 за ступенем важливості заголовки.

Синтаксис:

<h1> Заголовок 1-го рівня </h1>
 <h2> Заголовок 2-го рівня </h2>
 <h3> Заголовок 3-го рівня </h3>
 <h4> Заголовок 4-го рівня </h4>
 <h5> Заголовок 5-го рівня </h5>
 <h6> Заголовок 6-го рівня </h6>

Результат:

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня

Теги для форматування тексту


<strong> - зазначає, що його зміст має велике значення, серйозність або терміновість. Браузери зазвичай відображають вміст жирним шрифтом.

Синтаксис:

Браузери зазвичай відображають вміст <strong> жирним </strong> шрифтом.

Результат:

Браузери зазвичай відображають вміст жирним шрифтом.


<em> - призначений для акцентування тексту. Браузери відображають такий текст курсивним зображенням.

Синтаксис:

Браузери відображають такий текст <em> курсивним <em> зображенням.

Результат:

Браузери відображають такий текст курсивним зображенням


<small> - зменшує розмір шрифту на одиницю порівняно із звичайним текстом.

Синтаксис:

Браузери відображають такий текст <small> меншим </small>

Результат:

Браузери відображають такий текст меншим


<ins> - призначений для виділення тексту, доданого до нової версії документа.

Синтаксис:

Браузери відображають такий текст <ins>підкресленим</ins>

Результат:

Браузери відображають такий текст підкресленим


<del> - представляє діапазон тексту, який видалено з документа.

Синтаксис:

Браузери відображають такий текст як <del>видалено</del>

Результат:

Браузери відображають такий текст як видалено


<sub> - відображає шрифт у вигляді нижнього індексу.

Синтаксис:

H<sub>2</sub>O

Результат:

H2O


<sup> - відображає шрифт у вигляді верхнього індексу.

Синтаксис:

Крокет<sup>24</sup>

Результат:

Крокет24

Цитати,означення,абревіатури

<blockquote> - призначений виділення довгих цитат, які можуть складатися з кількох абзаців. Тег виділяє цитату як окремий блок тексту з відступами.

Синтаксис:

<blockquote>Ніколи нічого не просіть! Ніколи і нічого, і особливо у тих, хто сильніший за вас. Самі запропонують і самі все дадуть!</blockquote>

Результат:

Ніколи нічого не просіть! Ніколи і нічого, і особливо у тих, хто сильніший за вас. Самі запропонують і самі все дадуть!


<q> - призначений виділення коротких цитат в пропозиціях. Текст усередині цього тегу автоматично обрамляється лапками.

Синтаксис:

Відомий вираз <q>Hasta la vista, baby</q> належить Термінатору.

Результат:

Відомий вираз Hasta la vista, baby належить Термінатору.


<cite> - використовується для того, щоб виділити джерело цитати, назву твору чи автора цитати.

Синтаксис:

В книге <cite>Мастер и Маргарита</cite> есть такая цитата:

Результат:

В книге Мастер и Маргарита есть такая цитата:


<abbr> - показує, що послідовність символів є абревіатурою. За допомогою атрибуту title дається розшифровка скорочення, що дозволяє розуміти абревіатуру тим людям, які з нею не знайомі.

Синтаксис:

Ви можете використовувати <abbr title="Cascading Style Sheets">CSS</abbr>

Результат:

Ви можете використовувати CSS


<dfn> - використовується для вказівки терміна, визначаємо в контексті фрази або пропозиції

Синтаксис:

<dfn id="def-Валідатор ">Валідатор </dfn> – це програма, яка перевіряє синтаксичні помилки в коді або документах.

Результат:

Валідатор – це програма, яка перевіряє синтаксичні помилки в коді або документах.

Розмітка коду


<code> - призначений для відображення одного або декількох рядків тексту, який є програмним кодом.

Синтаксис:

<code> function checkParent (src, dest) {<Br> while (src != null) {<Br> if (src.tagName == dest) return src<Br> src = src.parentElement<Br> }<Br> return null<Br> }</code>

Результат:

function checkParent (src, dest) {
while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}

<kbd> -вказує на те, що текст всередині елемента описує введення користувача з клавіатури, голосове введення або введення з використанням іншого типу пристроїв введення тексту.

Синтаксис:

используйте комбинацию клавиш <kbd> <Alt>+<Backspace></kbd>

Результат:

используйте комбинацию клавиш <Alt>+<Backspace>


<samp> - використовується для відображення тексту, який є результатом виведення комп'ютерної програми чи скрипту.

Синтаксис:

<samp>Файл не знайдено.
Натисніть F1, щоб продовжити</samp>

Результат:

Файл не знайдено.
Натисніть F1, щоб продовжити

Списки


<OL> і <UL> - це службові теги, які потрібні тільки для того, щоб вказати браузеру, який саме вид списку ми формуємо,всередині яких знову ж таки будуть розташовані елементи <LI>

Синтаксис:

<ol>
<li>Мышь</li>
</ol>
<ul>
<li>Мышь</li>
</ul>

Результат:

  1. Мышь
  • Мышь

<img src="/img/icon1.png" alt="альтернативный">

Розриви

<br> - початок нового рядка (текст після тега починається з нового рядка);

Синтаксис:

початок нового рядка <br> (текст після тега починається з нового рядка);

Результат:

початок нового рядка
(текст після тега починається з нового рядка);


<hr> - горизонтальна лінія (у рядку, де розміщено тег, відображається горизонтальна лінія).

Синтаксис:

Глава №1 <hr> Вступ.Прийнятя

Результат:

Глава №1

Вступ.Прийнятя
